# Project Scope

## Project Objective

Analyze the end-to-end e-commerce customer journey to understand:

* Traffic acquisition
* Conversion performance
* Revenue generation
* Refund impact
* Customer retention

The project aims to generate actionable business insights and recommendations using SQL, Python, Statistics, and Power BI.

---

# In Scope

## Acquisition Analysis

* Traffic sources
* Campaign performance
* Device distribution

## Conversion Analysis

* Conversion rate
* Funnel drop-off
* Device performance

## Revenue Analysis

* Revenue
* Orders
* AOV
* Revenue trends

## Refund Analysis

* Refund rate
* Refund amount
* Net revenue

## Attribution Analysis

* First-touch attribution
* Last-touch attribution

## Retention Analysis

* Repeat visitors
* Cohort analysis
* Customer Lifetime Value (LTV)

---

# Out of Scope

* Real-time analytics
* Data engineering pipelines
* Data warehouse development
* Deep learning
* Production deployment

# Deliverables

* SQL Analysis
* Python Analysis
* Statistical Validation
* Power BI Dashboard
* Business Recommendations
